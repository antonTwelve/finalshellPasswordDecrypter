一个解密finalshell保存的密码的网页工具

用法：

打开finalshell安装目录下的conn文件夹，找到存储了对应密码的json文件，复制`password`内容，

打开右边链接或者下载本仓库所有文件后在浏览器打开

粘贴`password`内容到输入框，完事
